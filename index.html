What is Hack Club and Build Guild?

Hack Club

Hack Club is a global nonprofit community for teenage makers. It helps students learn by building real projects and running local events.

Build Guild

Build Guild is Hack Club Blueprint's 1-day hardware meetup format. Local organizers host hands-on sessions where teens can build, learn, and share projects together.
